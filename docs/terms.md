## 1. 服务说明

AxiomCode 提供 AI 模型的统一接入与中转服务，通过 OpenAI / Anthropic 兼容接口转发请求至上游模型。

## 2. 账户与密钥

- 你需注册账户并妥善保管 API 密钥；密钥泄露造成的损失由账户持有人承担。
- 建议为不同用途分别创建密钥并设置额度上限。

## 3. 计费与充值

- 采用预付额度、按量计费；具体倍率以控制台为准。
- 充值 / 兑换码相关规则（含是否可退）后续细化。

## 4. 使用规范

- 不得用于任何违法、侵权或滥用用途。
- 你需遵守所在地及相关司法管辖区的法律法规。

## 5. 服务范围

AxiomCode **不对中国大陆用户提供服务**。继续使用即表示你确认不属于受限区域，并自行承担合规责任。

## 6. 可用性与免责

服务按「现状」提供，可能因上游波动、维护等出现中断；在法律允许范围内，我们不对由此产生的间接损失负责。

我们可能更新本协议，更新后继续使用即视为接受。

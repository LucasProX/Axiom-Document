# 1.1 注册账号

> 目标：在 [axiomcode.top](https://axiomcode.top) 注册账号并完成邮箱激活。这是使用 AxiomCode 的第一步。

## 第一步：打开注册页

访问 **[https://axiomcode.top](https://axiomcode.top)**，点击右上角的「注册」按钮。

如果右上角已显示你的昵称 / 头像，说明你已经登录，可直接跳到 [创建 API 密钥](/guide/apikey/enter)。

![打开注册页](/images/prepare/01-open-register.png)

## 第二步：填写注册信息

按表单依次填写：

| 字段 | 说明 |
| --- | --- |
| **邮箱** | 用于登录、找回密码与账户通知，请使用常用、能正常收信的邮箱 |
| **密码** | 至少 8 位，建议字母 + 数字 + 符号组合 |
| **邀请码** | 选填 |
| **验证码** | 按提示完成 |

![填写注册信息](/images/prepare/02-fill-form.png)

::: warning 邮箱很重要
账户安全提醒、额度变动、密钥泄露预警都会发送到这个邮箱，请务必填写真实常用地址。
:::

## 第三步：提交并完成激活

点击注册后，到邮箱查收 AxiomCode 的激活邮件（如未收到，检查「垃圾邮件 / 推广」文件夹），点击邮件中的激活链接完成激活。

![提交注册](/images/prepare/03-submit.png)

## 第四步：登录并验证账号

回到 [axiomcode.top](https://axiomcode.top)，用刚才的邮箱与密码登录。登录成功后，右上角会显示你的头像 / 昵称，左侧菜单出现账户、计费与 API 密钥等入口。

![登录后界面](/images/prepare/05-after-login.png)

## 常见问题

**提示「该邮箱已注册」？** 说明此邮箱此前注册过，请用「找回密码」或换一个邮箱注册。

**收不到激活邮件？**

1. 检查「垃圾邮件 / 推广」文件夹。
2. 改用 Gmail / Outlook / 163 / 企业邮箱，QQ 邮箱偶尔会拦截。
3. 仍然没有，在登录页点击「重新发送激活邮件」。

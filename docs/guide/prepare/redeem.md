# 1.3 使用兑换码

> 兑换码是推荐的充值方式：**付款 → 获取码 → 粘贴兑换 → 立刻到账**，全程无需绑卡或填写敏感信息，也方便团队统一采购、分发。

## 第一步：购买兑换码

访问小店 [https://pay.ldxp.cn/shop/axiom](https://pay.ldxp.cn/shop/axiom)，选择面额，填入收码邮箱，扫码付款。成功后页面会直接显示兑换码，邮箱也会收到一份副本。

![小店首页](/images/prepare/08-shop-home.png)

![获取兑换码](/images/prepare/09-shop-code.png)

## 第二步：进入充值页

登录 [https://axiomcode.top/billing](https://axiomcode.top/billing)，找到页面中的「兑换码」区域。

![兑换码区域](/images/prepare/10-redeem-area.png)

## 第三步：输入并提交

将兑换码完整粘贴到输入框，点击提交。

![粘贴兑换码](/images/prepare/11-paste-code.png)

## 第四步：验证到账

刷新页面，在钱包记录中查看，类型显示为「最近兑换」即代表成功。

![兑换成功](/images/prepare/12-redeem-success.png)

![兑换记录](/images/prepare/13-redeem-record.png)

## 关键提醒

::: danger 兑换码等同现金
未兑换前任何拿到码的人都能使用，请妥善保管，不要公开分享或截图外发。
:::

- 兑换码**区分大小写**，需逐一兑换（不支持批量输入）。
- 首次使用建议先买**最小面额**走通流程。
- 提示「无效」通常是：字符缺失、大小写错误、或已被使用。兑换后若余额未更新，先刷新页面。

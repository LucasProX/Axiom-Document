# 常见问题 FAQ

## 账户与计费

**一个账户能创建多少把 API Key？**
通常没有硬性数量上限。建议：一设备 / 一项目 / 一成员各一把，CI / 自动化单独再开一把并设置额度上限。

**充值的钱会过期吗？**
取决于兑换码的条款。对于不限期的兑换码，兑换后进入账户余额，长期有效。

**可以开发票吗？**
AxiomCode 不直接开具发票，如有需要请联系客服特殊处理。

**怎么查消费明细？**
控制台的计费 / 账单区会展示「消费曲线 + 模型用量」，可按日期、Key 或模型筛选。

## API Key

**Key 泄露了怎么办？**
立即在密钥页停用 / 删除。更关键的是给所有 Key 都设「额度上限」—— 泄露的 Key 一旦触顶就会自动锁定，损失可控。

**多台机器能用同一把 Key 吗？**
可以。但多人共用同一把 Key **强烈不建议**，否则责任与用量都说不清。

**创建后还能改名称 / 额度 / 模型范围吗？**
可以，在密钥页点「编辑」即可修改。

## 客户端

**为什么 Codex 和 Claude Code 的 Base URL 不一样？**
协议不同：Codex 用 `https://axiomcode.top/v1`（OpenAI 兼容），Claude Code 用 `https://axiomcode.top`（Anthropic 兼容，**不带** `/v1`）。

**一把 Key 能同时用于 Codex 和 Claude Code 吗？**
可以。同一把 Key 在两种协议下都能鉴权，只要分别配上正确的 Base URL。

**能在官方账号和 AxiomCode 之间切换吗？**
可以，借助 `direnv` 等环境变量管理工具按目录切换即可。

**完整的模型 ID 列表在哪？**
控制台 → 密钥页 → 「接入配置」按钮，即可看到可用模型。也可参考 [支持的模型](/guide/models)。

## 网络与安全

**国内访问需要代理吗？**
不需要。关掉系统代理直连反而更快；Clash / V2Ray 等工具可能对请求做意外改写。

**请求内容会被记录吗？**
仅保留**元数据**（时间戳、token 用量、状态码）用于计费，不持久化请求体。

**AxiomCode 会自动续费吗？**
不会。它是**预付额度模式**，用完即停。

# 2.1 进入密钥页面

> 充值之后，下一步就是生成一把 **API Key** —— 所有客户端都用它来鉴权。本节先找到密钥管理页面。

## 第一步：打开控制台

登录 [https://axiomcode.top](https://axiomcode.top) 进入控制台仪表盘。

![控制台仪表盘](/images/apikey/01-dashboard.png)

## 第二步：找到 API 密钥菜单

在左侧菜单点击「API 密钥 / 令牌管理」项（图标通常是一把钥匙 🔑）。

![侧边栏菜单](/images/apikey/02-sidebar.png)

::: tip 侧边栏收起了？
在某些主题下，侧边栏会折叠成图标，鼠标悬停即可展开。
:::

## 第三步：查看密钥列表

进入后即是 API 密钥列表页。新账户通常会看到一个「创建密钥 / 新建 Key」按钮。列表一般包含这些列：

| 列 | 含义 |
| --- | --- |
| 名称 | 自定义备注，便于区分用途 |
| Key | `sk-xxxx` 字符串，**仅在创建时完整显示一次** |
| 状态 | 启用 / 停用 |
| 用量 / 额度 | 该 Key 的消费与上限 |
| 创建时间 | —— |
| 操作 | 编辑、停用、删除 |

![密钥列表](/images/apikey/03-key-list.png)

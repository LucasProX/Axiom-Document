## 我们收集的信息

- **账户信息**：注册邮箱、登录凭证。
- **计费与用量元数据**：时间戳、token 用量、状态码、所用模型等。

## 我们如何使用

用于提供与维护服务、计费结算、安全防护与故障排查。

## 关于请求内容

我们仅保留用于计费的**元数据**，不持久化你的请求体（提示词 / 回复正文）。详见 [常见问题](/guide/faq)。

## 第三方

- 支付与充值由相应支付服务商处理。
- GitHub / Google 等第三方登录由对应平台处理。

## 数据安全

我们采取合理的技术与管理措施保护数据，但任何传输与存储都无法保证绝对安全。

## 你的权利

你可在控制台查看用量、管理密钥；如需删除账户或导出数据，请通过 [联系我们](/contact) 与我们沟通。

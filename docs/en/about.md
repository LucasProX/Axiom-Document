# About Us

**AxiomCode** is a unified access layer for AI models — one key and one OpenAI / Anthropic-compatible endpoint that connects Claude, GPT, Gemini, Grok, and leading Chinese large models.

> In Euclidean geometry, an axiom is a proposition accepted as true without proof. AxiomCode brings that structure into the AI era: a single unified endpoint that makes every call identical from now on.

## What We Do

- **Aggregate frontier models**: bring Claude, GPT, Gemini, Grok, and Chinese large models together behind a single compatible interface.
- **Stable access**: optimized for China, low-latency direct connections, pay-as-you-go billing, and transparent usage.
- **Client-friendly**: support for mainstream AI coding clients such as Codex, Claude Code, OpenCode, and OpenClaw.

## What We Believe

Tools should be **identical** — whichever model you call and whichever client you use, developers should face the same clean, deterministic interface.
